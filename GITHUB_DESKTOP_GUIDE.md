# GitHub Desktop - Simple Guide

## First Time Publishing (Do This Once)

1. **Open GitHub Desktop**

2. **Add This Repository**
   - Click `File` → `Add Local Repository`
   - Click `Choose...`
   - Navigate to: `/Users/janorsag/Dropbox/SANDBOX_1/_EPK-Claude`
   - Click `Add Repository`

3. **If It Says "Not a Git Repository"**
   - Click `Create a Repository`
   - Repository name: `dj-yawn-epk`
   - Description: `DJ YAWN Electronic Press Kit 2026`
   - Click `Create Repository`

4. **Publish to GitHub**
   - Click the `Publish repository` button (top right)
   - **UNCHECK** "Keep this code private" (required for GitHub Pages on free plan)
   - Click `Publish Repository`
   - Done! Your code is now on GitHub

5. **Enable GitHub Pages**
   - Go to: https://github.com/YOUR-USERNAME/dj-yawn-epk
   - Click `Settings` tab
   - Click `Pages` in left sidebar
   - Under "Branch", select: `main`
   - Click `Save`
   - Wait 1-2 minutes
   - Your EPK is live at: `https://YOUR-USERNAME.github.io/dj-yawn-epk/`

## Making Updates (Every Time You Edit)

1. **Edit your files** (index.html or images)

2. **Open GitHub Desktop**
   - It will automatically show your changes

3. **Commit Your Changes**
   - Bottom left: Enter a short message (e.g., "Updated bio")
   - Click `Commit to main`

4. **Push to GitHub**
   - Top right: Click `Push origin`
   - Your changes will appear online in 1-2 minutes

## Common Questions

**Q: How do I see my changes online?**
A: Visit: `https://YOUR-USERNAME.github.io/dj-yawn-epk/`
   Wait 1-2 minutes after pushing for changes to appear.

**Q: I made changes but they're not showing online**
A:
   1. Check GitHub Desktop - did you commit and push?
   2. Wait 2-3 minutes (GitHub Pages needs time to rebuild)
   3. Hard refresh your browser (Cmd+Shift+R on Mac)

**Q: Can I delete the old dj-yawn-epk-web folder?**
A: Yes! Once this is working, the old folder is not needed.

**Q: What if I want to start over?**
A:
   1. Delete the repository on GitHub.com
   2. In GitHub Desktop, right-click the repo → "Remove"
   3. Follow "First Time Publishing" steps again
